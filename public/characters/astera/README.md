# Astera

`Astera.vrm` is a renamed copy of the Vita VRoid Studio sample model from the `madjin/vrm-samples` mirror of VRoid sample models.

The source model's embedded VRM metadata identifies the original title as `Vita`, allows use by everyone, allows commercial use, and lists the license as `CC0`. This repo uses the character name `Astera` for the bundled demo profile.

Source and license references:

- https://github.com/madjin/vrm-samples
- https://vroid.pixiv.help/hc/en-us/articles/4402614652569-Do-VRoid-Studio-s-sample-models-come-with-conditions-of-use
